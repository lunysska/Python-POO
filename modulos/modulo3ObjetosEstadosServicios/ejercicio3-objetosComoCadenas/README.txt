Este ejemplo muestra como se usa el __str__
1. Para mostrar los detalles del objeto teníamos el metodo "mostrarDetalles"
2. Si imprimimos un objeto, saca una referencia a la memoria
3. Entonces definimos el metodo __str__ que regresa una cadena
4. y ya podemos imprimir un objeto
5. cambiamos mostrarDetalles en todas las clases por este nuevo

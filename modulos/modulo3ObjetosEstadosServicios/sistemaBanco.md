# Sistema del Banco

La idea es ir practicando cada concepto teórico y aplicarlo al sistema del Banco, para que no sea pesado, cada concepto debe probarse y guardarlo en un directorio específico.
- Cada prueba se debe ir guardando en un directorio específico.

- Para el tema 3.1. El directorio debe llamarse **relacionesEntreObjetos**
- Para el tema 3.2. El directorio debe llamarse **condicionalesPD**
- Para el tema 3.4. El directorio debe llamarse **objetosComoCadenas**

Esta versión incluye el uso de lista de Cuenta en la clase Cliente, osea es la representación de 1 a muchos (Un cliente tiene muchas cuentas), para ello se debe hacer lo siguiente:


1. Agregar un atributo de listas en la clase Cliente, lo llamaremos en plural, por el manejo de muchos.
2. En el método constructor se debe inicializar adecuadamente a este atributo, en este caso va vacío, pero no siempre es así, depende de lo que se quiera hacer.
3. Se deben agregar los métodos que nos permitan manipular a estos objetos dentro de la lista, por ejemplo el método agregarCuenta, borrarCuenta, recuperarCuenta, etc. Los métodos que agregas a tu clase depende de lo que quieras hacer y se debe pensar muy bien los argumentos que cada método recibe.

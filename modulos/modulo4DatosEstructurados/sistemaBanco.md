# Sistema del Banco

La idea es ir practicando cada concepto teórico y aplicarlo al sistema del Banco, para que no sea pesado, cada concepto debe probarse por separado, en los archivos más actuales que tengas, y guardarlo en un directorio específico.

- Para el tema 4.1. El directorio debe llamarse **publicoVsPrivado** (se puede copiar y pegar el nombre)
- Para el tema 4.2. El directorio debe llamarse **listasDeCuentas**. Acá tambien debe actualizarse el menu, ya que ahora se debe considerar la relación Cliente-Cuenta.
